WODLOG App Icon Pack

Included files:
- wodlog_icon_master_1024.png : master icon PNG
- adaptive/ic_launcher_foreground.png : transparent foreground for Android adaptive icon
- adaptive/ic_launcher_background.png : solid background for Android adaptive icon
- adaptive/ic_launcher_monochrome.png : monochrome icon for themed icons
- android_legacy/mipmap-*/ic_launcher.png : legacy launcher icons
- android_legacy/mipmap-*/ic_launcher_round.png : round launcher icons
- common_sizes/wodlog_icon_256.png, 512.png, 1024.png : convenient export sizes

Suggested Android Studio mapping:
Foreground Layer -> adaptive/ic_launcher_foreground.png
Background Layer -> adaptive/ic_launcher_background.png
Monochrome Layer -> adaptive/ic_launcher_monochrome.png
Legacy/Round icons can be generated by Image Asset Studio or use the provided mipmap files.
