# 네이버카페 WOD 가져오기 QA 체크리스트

v0.2.0 네이버카페 WOD 가져오기 흐름의 수동 QA 항목이다. 이 기능은 사용자 주도 WebView 탐색과 저장 전 확인을 전제로 하며, 자동 크롤링이나 계정 정보 저장을 포함하지 않는다.

## CafeSource 설정

- [ ] Settings에서 `네이버카페 WOD 가져오기`로 진입할 수 있다.
- [ ] CafeSource를 추가, 수정, 삭제할 수 있다.
- [ ] Box 이름, 게시판 URL, 제목 키워드, 모바일 URL 우선 사용 값이 앱 재실행 후에도 유지된다.
- [ ] 빈 Box 이름, 빈 URL, 잘못된 URL 입력 시 한국어 오류가 표시된다.

## Home 진입점

- [ ] CafeSource가 없으면 HomeScreen의 `WOD 불러오기` 진입점이 숨김 또는 비활성화된다.
- [ ] CafeSource가 1개 있으면 HomeScreen에서 바로 WOD 불러오기 화면으로 진입한다.
- [ ] CafeSource가 2개 이상 있으면 가져올 카페 선택 UI가 표시된다.
- [ ] 기존 `WOD 추가` 흐름은 수동 입력으로 그대로 동작한다.

## WebView 가져오기 화면

- [ ] WebView가 등록한 게시판 URL을 연다.
- [ ] 현재 URL, 로딩 상태, 새로고침, WebView 뒤로가기가 표시되고 동작한다.
- [ ] WebView 로딩 실패, 네트워크 없음, 잘못된 URL에서 한국어 오류와 새로고침/URL 확인 안내가 표시된다.
- [ ] 로그인 필요한 카페는 사용자가 WebView 안에서 직접 로그인할 수 있다.
- [ ] 앱은 네이버 ID/PW, 쿠키, 세션, 토큰을 별도로 요구하거나 저장하지 않는다.

## 실패 및 fallback

- [ ] 삭제된 CafeSource id 또는 잘못된 가져오기 route 접근 시 crash 없이 설정 확인 안내가 표시된다.
- [ ] CafeSource가 없는 가져오기 화면에서 설정 이동, 수동 WOD 추가, 뒤로가기 동작이 가능하다.
- [ ] 후보가 없거나 후보 JS 결과 파싱에 실패하면 한국어 empty/fallback 문구가 표시된다.
- [ ] 본문이 너무 짧거나 iframe 구조 등으로 추출 실패 시 한국어 오류/fallback 문구가 표시된다.
- [ ] 미리보기 데이터가 비어 있는 route 접근 시 crash 없이 다시 가져오기 또는 수동 입력 안내가 표시된다.
- [ ] imported prefill holder가 비어 있는 WOD Edit route 접근 시 일반 WOD 입력으로 시작한다는 안내가 표시된다.

## 후보 및 본문 추출

- [ ] `현재 목록에서 WOD 찾기`를 누르기 전에는 후보 추출이 실행되지 않는다.
- [ ] 후보 추출 결과가 있으면 제목, URL, 날짜 또는 키워드 정보가 목록에 표시된다.
- [ ] 후보 선택 시 해당 글이 WebView 안에서 열린다.
- [ ] `본문 가져오기`를 누르기 전에는 본문 추출이 실행되지 않는다.
- [ ] 본문 추출 성공 시 `가져온 WOD 미리보기` 화면으로 이동한다.

## 미리보기와 WOD Edit 적용

- [ ] 미리보기 화면에 source URL, 제목, 가져온 본문이 표시된다.
- [ ] 미리보기에서 `WOD 입력에 적용`을 누르면 WOD Edit 화면으로 이동한다.
- [ ] WOD Edit에서 제목, 날짜, 원문을 사용자가 수정할 수 있다.
- [ ] recomposition이나 화면 갱신으로 사용자가 수정한 값이 prefill 값으로 덮이지 않는다.
- [ ] 사용자가 WOD Edit 저장 버튼을 누르기 전에는 WOD가 DB에 저장되지 않는다.
- [ ] 저장 후 WOD 목록/상세에서 기록을 확인할 수 있다.
- [ ] source metadata가 백업 export/import에서 유지된다.

## 회귀 및 보안 확인

- [ ] 일반 WOD 추가, 기존 WOD 상세, 결과 입력, 백업 export/import 흐름이 기존처럼 동작한다.
- [ ] 네이버 ID/PW, 쿠키, 세션, 토큰 입력 UI 또는 저장 필드가 없다.
- [ ] 자동 반복 추출, 페이지 자동 순회, 백그라운드 수집, CAPTCHA 우회, 차단 회피 동작이 없다.
- [ ] 서버, Firebase, Supabase, OpenAI API, 네이버 API 연동이 추가되지 않았다.
