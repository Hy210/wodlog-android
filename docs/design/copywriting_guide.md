# 와드로그 문구 작성 가이드

이 문서는 와드로그 앱에 표시되는 한국어 문구와 CrossFit 용어 표기 기준을 정의한다.

## 1. 전체 문구 원칙

- 앱에 출력되는 안내 문구, 버튼, 오류 메시지, 빈 상태, 설명 문구는 한국어로 작성한다.
- 존댓말을 사용한다.
- 짧고 명확하게 쓴다.
- 사용자를 탓하지 않는다.
- 운동 직후 빠르게 이해할 수 있는 표현을 쓴다.
- 저장, 복사, 추가, 삭제 같은 핵심 액션은 명확히 구분한다.
- 삭제, 초기화, Import 덮어쓰기처럼 데이터 손실 가능성이 있는 액션은 되돌릴 수 있는지 여부를 분명히 알린다.

## 2. 한국어 안내 문구 정책

- 기술 용어를 불필요하게 노출하지 않는다.
- “에러”, “Exception”, “DB” 같은 구현 용어 대신 사용자가 할 수 있는 행동을 안내한다.
- 한 문장은 가능하면 30자 안팎으로 유지한다.
- 긴 설명이 필요하면 제목과 보조 설명을 분리한다.

좋은 예:

- 기록을 불러오지 못했습니다
- 다시 시도해 주세요
- 입력값을 확인해 주세요

피할 예:

- DB 조회 실패
- 잘못된 값을 입력했습니다
- Exception이 발생했습니다

## 3. CrossFit 용어 영어 유지 정책

CrossFit에 익숙한 WOD 카테고리, 운동 종류, 기록 유형, CrossFit 용어는 영어 표기를 유지한다.

영어로 유지할 용어:

- For Time
- AMRAP
- EMOM
- RFT
- Chipper
- Strength
- Skill
- Metcon
- Rx
- Scaled
- Custom
- Thruster
- Pull-up
- Deadlift
- Row
- Wall Ball
- Toes-to-bar
- Burpee
- Clean
- Snatch
- RPE

한국어 설명이 필요한 경우 영어 뒤에 보조 설명을 붙인다.

- AMRAP - 제한 시간 동안 최대 라운드
- Rx - 지정된 기준으로 수행
- Scaled - 조정된 난이도로 수행
- For Time - 정해진 작업을 빠르게 완료

CrossFit 용어를 억지로 한국어 번역하지 않는다. 같은 용어는 앱 전체에서 동일하게 표기한다.
